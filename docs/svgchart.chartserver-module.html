addXml(root, path, value)

source code 

Adds an XML node or attribute to a tree of Elements.

Parameters:
  • root - The root of the tree of Elements
  • path - The path to the node or attribute. /elem/elem, /elem@attr, /elem@style$sty, etc.
  • value - The value to be stored at path. May be inner text, attribute value, or style value depending on path.
