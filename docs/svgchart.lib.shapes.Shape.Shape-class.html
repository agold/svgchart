Base class of a generic shape.

>>> shape = Shape(tag='foo', id='id', classes='class', text='bar', attrs={'spoons': 'forks'}, knives='sharp')
>>> shape.tag
'foo'
>>> shape.text
'bar'
>>> shape.attrs
{'spoons': 'forks', 'knives': 'sharp'}
>>> print shape
<foo class="class" id="id" knives="sharp" spoons="forks">bar</foo>
